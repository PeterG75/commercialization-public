# <span id="p_hlk_test.e9ec5310-8e8e-42d8-ab80-594511d561c9"></span>Ambient Light Sensor Functional Verification Tests


Validates the functionality of Ambient light sensor device i.e. validating the data notifications for different sampling intervals and validating the steady state of successive samples.

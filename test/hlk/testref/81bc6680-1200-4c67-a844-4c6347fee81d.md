# <span id="p_hlk_test.81bc6680-1200-4c67-a844-4c6347fee81d"></span>Gyrometer Sensor Basic Validation Tests


Validates basic operations on 3D Gyrometer sensor device such as open/close handles, start/stop data notification, retrieving sensor data, and querying sensor capabilities
